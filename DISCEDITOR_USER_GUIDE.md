## 📋 使用说明

### 第一次使用

1. **刷新浏览器页面**
   - 按 `Ctrl + F5` 强制刷新，确保加载最新代码

2. **点击种子名称**
   - 在主页面的种子列表中点击任意种子名称
   - DiscEditor 对话框会弹出

3. **查看文件树**
   - 第一次打开时，系统会自动从 qBittorrent 同步文件（可能需要几秒钟）
   - 文件树会显示完整的目录结构和文件列表
   - 每个文件会显示大小

4. **分配卷号**
   - 在目录或文件右侧的下拉框中选择卷号（第1卷、第2卷...）
   - 选择目录会自动包含其下所有文件
   - 系统支持最多 20 卷

5. **填写卷信息**
   - 在"卷信息"区域填写每卷的：
     - 品番（catalog_no）
     - 卷名（volume_name）

6. **保存**
   - 点击"保存"按钮
   - 数据会保存到数据库
   - 下次打开同一种子时会自动加载已保存的卷分配

### 后续使用

- **查看已分配的卷**：种子列表中的"卷"列会显示已分配的卷数
- **编辑已有卷**：再次点击种子名称，会加载已保存的卷分配信息
- **同步最新文件**：如果 qBittorrent 中的文件有变化，可以使用"同步光盘文件"功能

## 🔍 调试信息

如果遇到问题，可以查看浏览器控制台（按 F12）：

- `[DiscEditor] 从 qBittorrent 同步文件...` - 表示正在同步
- `[DiscEditor] 加载文件: X 个` - 显示加载的文件数
- `[DiscEditor] 没有获取到文件数据` - 表示获取文件失败

## 📊 技术细节

### 数据流程

1. 点击种子名称 → 触发 `open()` 函数
2. 尝试从数据库获取文件（`/api/torrents/files`）
3. 如果数据库为空，从 qBittorrent 同步（`/api/qb/torrents/files`）
4. 构建文件树结构
5. 如果有已保存的卷数据，加载并显示

### 数据格式

FileItem 结构：
```typescript
{
  id: string          // 文件唯一 ID
  name: string        // 文件路径/名称
  size: number        // 文件大小（字节）
  progress: number    // 下载进度 (0-1)
}
```

---

### 1. 核心光盘镜像结构 (The Disc)

这是资源的主体，通常以文件夹形式存放。

#### **Blu-ray (BDMV) 结构**

* **`BDMV/`** (核心文件夹)
* **`STREAM/`**: **最重要的目录**。包含所有视频流（`.m2ts`）。
* *体积最大的文件* 通常是正片（Feature）。
* *较小的文件* 通常是 Menu 动画、OP/ED 或短篇特典。


* **`PLAYLIST/`**: `.mpls` 播放列表。定义了视频段落的组合逻辑（如：播放正片时是否跳过广告）。
* **`CLIPINF/`**: `.clpi` 数据库文件。存储视频流的属性信息。
* **`BACKUP/`**: 上述关键文件的备份。


* **`CERTIFICATE/`**: 包含 AACS 授权证书，部分专业播放器（如海美迪、OPPO）需要此文件夹才能正常导航菜单。

#### **DVD (VIDEO_TS) 结构**

* **`VIDEO_TS/`**
* **`.VOB`**: 实际的视频数据（每段上限 1GB）。
* **`.IFO`**: 导航信息（菜单、音轨切换、字幕设置）。
* **`.BUP`**: IFO 文件的备份。



---

### 2. 附加内容结构 (Extra / Scans)

日本动画原盘往往附赠精美的物理周边，PT 组通常会将这些内容“数字化”后放在镜像根目录之外。

#### **Scans / Booklets (扫图)**

* **`Scans/`**: 包含以下内容（通常为高分辨率的 `.png` 或 `.tiff`）：
* **Covers / Jackets**: 包装盒封套。
* **Booklets**: 随盘附赠的小册子、设定集、分镜稿摘录。
* **Discs / Labels**: 光盘盘面的直接扫描。
* **Postcards / Stickers**: 随盘附赠的明信片或贴纸。



#### **CD / Music (特典 CD)**

日本动画首发版（Limited Edition）常附赠 CD。

* **`CD/`** 或 **`Music/`**:
* **OST (Original Soundtrack)**: 原声带。
* **Drama CD**: 广播剧。
* **Character Song**: 角色歌。
* *注：通常包含 `.flac` 无损音频、`.cue` 列表文件以及该 CD 的独立扫图。*


## 日本动画物理媒介发行体系摘要

### 1. 核心发行形态 (Release Formats)

* **分卷发行 (Individual Volumes/Single Volumes):**
* **定义：** 动画首播期间或完结后不久，按顺序分批次发行。
* **规格：** 通常每卷包含 2-3 集（24分钟/集）。一个季番（12-13集）通常分为 4-6 卷。
* **内容：** 包含本篇映像、限定特典（如 CD、手册、优先购票券）。


* **分卷合集 (Part Sets / Half-Season Sets):**
* **定义：** 常见于海外授权版（如北美），将一季动画分为 Part 1 和 Part 2 发行。


* **BOX / 完整盒装 (Box Sets):**
* **定义：** 将整部作品或整个季度打包在一个硬盒内。
* **分类：** * **Standard Edition:** 仅含碟片。
* **Limited Edition (完全生产限定版):** 包含大量周边特典。




* **BOX 合集 (BOX Sets Collection / Memorial BOX):**
* **定义：** 针对长篇动画（如《龙珠》、《海贼王》）或多季作品，将多个 BOX 再次整合。



---

### 2. 版本与编码规范 (Versions & Coding)

| 术语 | 定义与特征 |
| --- | --- |
| **初回生产限定版** | 第一批印刷版本，通常附带最高规格的特典（如原画集、活动抽选券），售罄后不再生产。 |
| **通常版 (Regular Edition)** | 仅包含碟片和基础内页，长期供应。 |
| **廉价版 (Budget Re-release)** | 在作品发行数年后，以极低价格再次推出的版本（如“EMOTION the Best”系列）。 |
| **JAN码 (JAN Code)** | 日本商品条码（13位），是识别不同版本、不同店铺限定的核心索引。 |
| **规格品番 (Catalog Number)** | 由发行商定义的唯一代码，如 **ANZX-XXXX**（BD限定版）、**ANSX-XXXX**（BD通常版）。前四位字母通常代表发行公司和介质。 |

---

### 3. 特典体系 (Bonus Content)

* **封入特典 (Enclosed Bonuses):** 包装盒内的物理物品（小册子、原声带 CD、特制卡片）。
* **映像特典 (On-disc Bonuses):** 碟片内的数字内容（无字幕 OP/ED、声优访谈、未放送片段、修正后的作画）。
* **店铺特典 (Store-specific Bonuses):** 针对不同销售渠道（如 Animate, Amazon, Rakuten）设计的专属赠品（挂画、文件夹、音轨）。
* **全卷购入特典:** 收集全套分卷后，凭兑换券向官方申领的终极奖励（通常是一个收纳盒或特别番外篇）。

---

### 4. 行业关键术语 (Key Industry Terms)

* **修补作画 (Retakes):** BD/DVD 版通常会对电视放送（TV Version）中的崩坏作画、光效遮盖（圣光/暗牧）进行修正或解除。
* **二卷销量 (Vol. 1 Sales):** 行业惯例中，第一卷的销量（通常称为“壁”）被视为衡量作品商业成功与否的核心指标。
* **POS数据:** 零售端的实时销售数据，用于统计销量排行榜（如 Oricon 榜单）。

### 1. Aniplex (核心前缀：A)

Aniplex 的编号逻辑非常严密，通过前四位字母即可判定介质类型、版本属性及价格带。

#### **常见的规格品番 (Catalog Number) 解析：**

* **ANZX：** **BD (Blu-ray) 完全生产限定版**。这是 Aniplex 最核心的盈利产品，通常包含 CD、画册等大量实体特典。
* **ANSX：** **BD 通常版**。仅包含碟片和基础内页，通常在限定版售罄后或为了压低售价时推出。
* **ANZB：** **DVD 完全生产限定版**。在流媒体时代依然保留 DVD 格式，照顾特定用户群体。
* **ANSB：** **DVD 通常版**。
* **ANXC / ANZC：** 通常用于非卖品、店铺特典碟或特定活动赠送的 BD/DVD。

#### **发行逻辑特点：**

* **全卷收纳 Box (Storage Box)：** Aniplex 擅长通过“全卷购入特典”赠送由原案/人设绘制的精美外盒，将散装的分卷整合成一个视觉整体。
* **价格溢价：** 相比其他厂商，Aniplex 的限定版单价通常较高，但特典内容（尤其是广播剧 CD 和原声带）的含金量极高。

### 2. Bandai Namco Filmworks (品牌名：EMOTION / 前缀：B)

万代（尤其是旗下的 Emotion 标签）历史悠久，编号体系经历过多次变革，目前的逻辑以介质和项目属性为主。

#### **常见的规格品番 (Catalog Number) 解析：**

* **BCXA：** **Blu-ray Disc**。这是目前最通用的万代蓝光编号（A 代表较高规格或一般流通）。
* **BCBA：** **DVD Video**。万代标准的 DVD 编号。
* **BCXJ / BCBJ：** 其中的 **'J'** 通常代表 **“廉价版”或“再发版”**。例如经典的“Visual Memories”或“EMOTION the Best”系列，售价通常只有初版的一半甚至更低。
* **BCXE：** 有时用于含有特殊联动或活动限定的 BD 规格。

#### **发行逻辑特点：**

* **B-Visual 策略：** 万代非常注重“旧作复刻”。他们拥有庞大的 IP 库（如《高达》、《攻壳机动队》），会定期推出 **Memorial BOX**（纪念盒装）或 **4K UHD Remaster** 版。

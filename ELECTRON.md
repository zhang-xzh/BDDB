# BDDB Electron 桌面应用

## 架构

```
┌─────────────────────────────────────────┐
│           Electron 应用                  │
│  ┌─────────────────────────────────┐    │
│  │         渲染进程                 │    │
│  │     (Chromium 前端页面)          │    │
│  └─────────────────────────────────┘    │
│                   ↑                     │
│  ┌─────────────────────────────────┐    │
│  │         主进程                   │    │
│  │   ┌─────────────────────────┐   │    │
│  │   │   Next.js 服务 (3000)   │   │    │
│  │   │   - API routes          │   │    │
│  │   │   - 页面渲染             │   │    │
│  │   └─────────────────────────┘   │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

## 开发模式

```bash
# 同时启动 Next.js dev server 和 Electron
npm run electron:dev
```

## 生产构建

```bash
# 构建独立应用（包含前后端）
npm run electron:build
```

输出：`dist/BDDB-Setup-x.x.x.exe`

## 打包内容

- **前端**：Electron 渲染进程
- **后端**：Next.js standalone + API routes
- **资源**：MongoDB/MeiliSearch 需外部运行

## 注意事项

1. **数据库**：MongoDB 和 MeiliSearch 仍需单独部署
2. **端口**：默认使用 3000，如被占用会自动检测
3. **数据持久化**：数据存储在 Docker 或外部数据库中

## 仅测试打包（不生成安装程序）

```bash
npm run electron:dir
```

输出到 `dist/win-unpacked/`，可直接运行 exe 测试
